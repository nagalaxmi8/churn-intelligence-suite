🚀 Customer Churn Prediction & Retention Intelligence System

An end-to-end team-based machine learning system that predicts customer churn using multiple models and generates actionable retention strategies powered by explainable AI.

📌 Overview

This project is a collaborative mini-project (team of 4) where each member independently built a churn prediction model, and all models were integrated into a unified intelligent system.

The system goes beyond basic prediction by combining:

🔮 Multiple ML Models (ensemble-style comparison)
📊 Explainable AI (SHAP)
💡 Automated Retention Recommendations
📈 Interactive Dashboard for decision-making

👉 Goal: Help businesses proactively identify high-risk customers and reduce churn with data-driven strategies.
